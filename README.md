# Inferno

_A [docker](//docker.com) [composition](//docs.docker.com/compose) pairing an instance of [Grotto](//github.com/daetal-us/grotto)_ with an instance of [Postgres](//postgres.org)

## Usage

```bash
docker-compose up
```
